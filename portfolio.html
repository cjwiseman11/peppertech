ZIKI.FM

ZIKI.FM is a project I created whilst learning React. It's a full MERN stack, using MongoDB for all the data storage, Express on the nodeJS server and React for the front-end.

Key Points

  • Automatic data gathering utilising the MusicBrainz database when a new artist is added
  • A queue system to gather new data when users visit artist pages
  • CRON runs to gather news articles once an hour
  • Automatically checks for new articles to display to the user
  • Making use of APIs from Facebook, Twitter, YouTube, Setlist.fm, Bandsintown and Last.fm

feedify.xyz

feedify.xyz was a second iteration of an RSS reader that was rebuilt using NodeJS Express server and Bootstrap front-end.

Key Points

  • Automatically gather news every 15 minutes using CRON
  • Uses Smmry API to provide a short summary for each article
